## What is Cardano N2N Client?

A simple(r) Cardano node-to-node client library in Haskell, wrapped around the [cardano-node](https://github.com/intersectmbo/cardano-node) libraries.

This library allows you to connect to a Cardano node using the node-to-node protocol and perform chain synchronization.
