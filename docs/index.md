# Cardano N2N Client

[![CI](https://github.com/paolino/cardano-n2n-client/actions/workflows/CI.yaml/badge.svg)](https://github.com/paolino/cardano-n2n-client/actions/workflows/CI.yaml)

A simple(r) Cardano node-to-node client library in Haskell.

This library allows you to connect to a Cardano node using the node-to-node protocol and perform chain synchronization.
